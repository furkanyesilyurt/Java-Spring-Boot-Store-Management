# Java-SpringBoot-PostgreSQL
This project is a small store management project made with spring boot.

## Project Structure
* Entity
* DTO
* DAO
* Service
* Controller
* Exceptions

## Built With
* Java
* Hibernate
* JPA
* Spring Boot
* PostgreSQL
